# Snipet

## Comandos

### Criação projeto
- NestJs:
    ```
    npx nest new <project-name> ou nest new <project-name>
    npx nest g module <module-name>
    npx nest g service <service-name>
    npx nest g controller <controller-name>
    ```

- Prisma:
    ```
    npx prisma init
    npx prisma migrate dev
    ```

- MySQL:
    ```
    mysql -u<user> -p<pass>
    use <database-name>;
    show tables;
    describe <table-name>;
    truncate <table-name>;
    ```
